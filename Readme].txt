The model is for the detection of objects(in this case the number of peoples) from the live video .
The model use Yolov3 which is a pretrained model ,along with the colab enviroment.
To use this model You have to download the essential file provided and run the code in the colab enviroment.
The code will work only on colab and not on any other enviroment.
You can use the code on other enviroment by makinn some changes.You can also use live video for the demonstration.
